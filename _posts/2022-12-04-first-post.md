### Printing hello world in different languages

#### Python

```py
print("hello world!")
```

#### JavaScript

```javascript
console.log("hello world!");
```

#### C

```c
include <stdio.h>

int main()
{
    printf("hello world!\n");
}
```

#### C++

```c++
include <iostream>

using namespace std;

int main()
{
    cout << "hello world!\n";
}
```

#### Java

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("hello world!");
}

}
```

#### Rust

```rust
fn main() {
    println!("Hello World!");
}
```

#### Ruby

```ruby
puts "hello world"
```

#### Swift

```swift
print("hello world!")
```

#### Go

```go
package main
import "fmt"
func main() {
    fmt.Println("hello world")
}
```
